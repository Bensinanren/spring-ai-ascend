阿里云百炼平台 模型定价

本页列出阿里云百炼（Bailian）大模型服务平台所提供的通义千问系列模型的计费方式与 公开价格。百炼平台按 token 用量计费，输入与输出分别计价，价格以人民币元每千 token 为单位，最终费用以账单明细为准。

计费模式

百炼平台提供按量付费与预付费资源包两种计费模式。按量付费无需预购，按实际调用 token 用量结算；预付费资源包以固定金额购买一定 token 额度，可在有效期内抵扣用量。两种模式 可叠加使用，系统优先抵扣资源包额度，超出部分按量计费。

qwen-max 价格

qwen-max 为通义千问旗舰大语言模型，适用于复杂推理与长文本理解。输入价格 0.04 元 / 千 token，输出价格 0.12 元 / 千 token，上下文窗口 32K。

qwen-plus 价格

qwen-plus 为通义千问均衡模型，在效果与成本之间取得平衡。输入价格 0.0008 元 / 千 token，输出价格 0.002 元 / 千 token，上下文窗口 128K。

qwen-vl-max 价格

qwen-vl-max 为通义千问视觉多模态模型，支持图像理解与图文对话。输入价格 0.02 元 / 千 token，输出价格 0.06 元 / 千 token。
