火山引擎方舟大模型平台 定价说明

本页说明火山引擎方舟大模型平台（Volcengine Ark）所提供的大语言模型与多模态模型的 计费方式与公开价格。方舟平台统一按 token 用量计费，输入与输出分别计价， 所有价格以人民币（元）每千 token 为单位，实际费用以账单为准。

计费方式概述

方舟平台采用按量付费的计费模式，无需预购套餐即可调用各模型。对于调用频次较高的企业客户， 也提供预付费资源包与承诺消费折扣。token 用量由平台统一计量，输入 token 与输出 token 分别按照对应单价结算，调用失败不产生费用。

Doubao-1.5-pro-32k 价格

Doubao-1.5-pro-32k 为通用大语言模型，上下文窗口 32K。输入价格为 0.0008 元 / 千 token， 输出价格为 0.002 元 / 千 token。该模型适用于文本理解、摘要、改写与一般问答场景。

Doubao-1.5-vision-pro 价格

Doubao-1.5-vision-pro 为多模态视觉理解模型，支持图像与文本联合输入。输入价格 0.003 元 / 千 token，输出价格 0.005 元 / 千 token；图片按其 token 化后的用量计入输入。

免费额度与限制

新开通方舟平台的企业账号可获得每个模型每日一定量的免费体验额度，超额部分按上表价格计费。 免费额度不可累积、不可转让，具体数值以控制台公示为准。
